This plugin is only available if 'compatible' is not set.

INTRODUCTION                                    *swapnumbers*

Swap numbers on your keyboard with the symbol they correspond to when shifted
in insert mode. This can make your coding a bit easier as you will not have to
use hold shift to use symbols so common in programming languages.

ABOUT                                           *swapnumbers-about*
